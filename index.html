How to use

  1. Input texts in the left textbox
  2. Specify how long is the series
  3. Click the "Generate" button
  4. The generated text will be output in the right textbox

Adding variables

Variables can be added with their initial value and how they change.
The format to add a variable is:
$ =
changed by
Value before changes

They can be used with a "$" head in the texts, and the values automatically substitute where variables are at.
Ex. when $1 is the default: 
$1 - $1 = 0
will be 
1 - 1 = 0

The "changed by" when adding a variable specifies a function or a process.
The function or process must be in the form of: 
$1 + 1
makes $1 increase by 1.
$1 * 1.5
makes $1 multiply by 1.5.
$1 + $2
makes $1 increase by $2 (if variable named "2" doesn't exist, this makes an error). This can also make a Fibonacci sequence represents by [1,1,2,3,5,8,...].
$1 * Math.random()
also can be used, but this is not recommended due to using JavaScript's functions. Only skillful users (such as programmers) should use advanced functions like this.

And spaces are needed around the variables in the "changed by" (not needed in the input text).
Ex.: 
$1+2
doesn't work as you've wanted and will make an error as referring variable named "1+2" (unless "1+2" exists, but it can't be made usually).

Removing and Rearranging variables

Variables can be removed by clicking the button.
Variables can be rearranged by clicking the buttons:
  • moves the variable on top
  • makes the variable ascend
  • makes the variable descend
  • moves the variable on bottom
Rearranging variables is for changing the order of the processes.

Variables can be prevented from being substituted for values the other one's included its name in the variables' name.
Ex.: 
$11
will not be substituted for $11's value if $11 comes after $1 on the processing order (unless $1's value was "$11" and such).

Another ex.: 
$$11
can be substituted for a value recursively as referring to $($1's value)1, if $1 comes before $11. This allows referring to variables by a linear search like [$11,$21,$31,...].
